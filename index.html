O que é?

Porque estudar Python?

É uma linguagem de programação de alto nível, possui uma forma específica de organizar softwares onde os procedimentos estão submetidos às classes, o que possibilita um maior controle e estabilidade de códigos para projetos de grandes proporções.
Uma linguagem de sintaxe relativamente simples e de fácil compreensão.


Para que é usado o Python?

o Python é uma linguagem muito popular nas áreas da tecnologia relacionadas à análise de dados, pesquisa, desenvolvimento de algoritmos e IA.

Lógica de Programação

A lógica de programação é a maneira como se escreve um algoritmo, que nada mais é do que uma sequência de passos necessários, para que uma função seja executada.

Em sua logica de programação o Python apresenta comandos básicos tais como:

Print () pula para a próxima linha
Input () aguarda o desenvolvedor digitar algo (valor) , com uma sequência de caracteres
Zip () empacota uma combinação sequencial em listas
Enumerate () enumera um objeto ou retoma a sua contagem
Range ()cria uma linha sequencial de números
Help () oferece ajuda em determinadas situações
Id () retoma a identidade de um objeto
Dir () retoma os métodos de um objeto
Str () converte um objeto em texto
Int () converte um objeto em número inteiro
